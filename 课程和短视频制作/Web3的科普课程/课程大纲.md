# Web3 入门与实战：8 小时小白友好课程大纲（精简版）

## 课程结构（8×1 小时）

### 第1课 Web3/区块链极简认知 + 钱包与安全
- 目标：理解“状态改变”的本质，完成钱包创建与安全备份。
- 理论（简明）：公私钥/地址与 EOA vs 合约账户；交易/区块与共识；Gas 与费用结构；L1/L2、链 ID 与最终性；托管 vs 自托管的权责。
- 实操：安装 MetaMask；创建 1 个账户；离线抄写助记词并验证；添加 `Sepolia`；签一条消息。
- 工具：MetaMask、Chainlist。
- 作业：写下你的备份策略与地址；在 `Sepolia` 转 0.001 ETH 给同学。

### 第2课 区块浏览器与交易细节
- 目标：看懂一笔交易与常见失败原因，能自查成本与风险。
- 理论（简明）：交易结构（from/to/value/data）；状态改变 vs 事件 Logs；内部交易（Internal Tx）；失败原因（Out‑of‑Gas/Revert）；手续费估计与优先费；区块时间与最终性。
- 实操：在区块浏览器查看一笔交易与事件；定位失败原因（如 Out‑of‑Gas 或 Revert）。
- 工具：Etherscan、BaseScan、Polygonscan。
- 作业：截屏并用 3 句话解释一笔交易的关键字段（Gas/nonce/事件）。

### 第3课 代币与稳定币 + NFT速讲（合并）
- 目标：掌握 ERC‑20/稳定币的授权与撤销，快速理解 NFT 的用途。
- 理论（简明）：ERC‑20 要点（总供给/小数位/授权额度）；稳定币模型（法币足额、超额抵押、算法）与锚定/赎回；NFT 的元数据/存储（IPFS/Arweave）、版税与二级市场。
- 实操：在钱包添加 USDC；小额转账；用 Revoke.cash 查看并撤销 1 条授权；（可选）在 OpenSea 查看一个 NFT 的 metadata。
- 工具：代币合约地址、Revoke.cash、DeBank、OpenSea（查看，可选）。
- 作业：列出你钱包的前 3 个授权并撤销 1 个；（可选）完成一次 NFT 转赠。

### 第4课 交易基础：CEX vs DEX 与下单机制
- 目标：会在 CEX/DEX 进行小额交易，理解成本与滑点。
- 理论（简明）：订单簿/做市与费用结构；AMM 定价与价格影响；滑点/流动性深度/价差；托管风险与提现通道；MEV/前置交易与交易税；路由与聚合器。
- 实操：连接 Uniswap 或 Jupiter；设置滑点；用 1 USDC 完成一次小额 Swap；CEX 仅演示界面，不做实操。
- 工具：币安/OKX/Coinbase、Uniswap/Jupiter、Coingecko（价格参考）。
- 作业：用 1 句话解释滑点与价格影响；记录本次交易的费用与执行价差。

### 第5课 交易指标与风控（市值/FDV/24h量/深度/持币分布）
- 目标：能用核心指标评估一个币的可交易性与风险。
- 理论（简明）：市值/FDV/流通量与解锁节奏；24h 量与真实流动性；深度/价差与可成交性；持币集中度与鲸鱼风险；TVL 与协议健康；异常信号（交易税/黑名单）。
- 实操：选一个代币，查询市值/FDV 与 24h 量；写出 2 条风控结论。
- 工具：Coingecko、CoinMarketCap、DexScreener、DeFiLlama、TradingView。
- 作业：提交一页评估卡（指标数据、结论与是否参与的理由）。

### 第6课 DeFi 入门：存借、收益与风险控制（含入金出金与跨链）
- 目标：体验借贷协议与资产流动的完整路径，理解清算与稳健收益边界。
- 理论（简明）：利率模型（利用率/可变/稳定利率）；健康度/清算机制与惩罚；抵押资产选择与波动性；最小授权原则；官方桥 vs 第三方桥安全；跨链消息 vs 资产转移；费用组成与等待时间。
- 实操：在 Aave/Compound 仅做少量存款；查看健康度与 APY；演示一次官方桥跨链（Base↔Ethereum）；不做入金/提现实操，仅说明流程与成本。
- 工具：Aave、Compound、Risk Dashboard、币安/OKX、MoonPay、Base Bridge、Polygon Bridge。
- 作业：用 2 条建议说明如何避免被清算（仓位控制/通知/质押资产选择）；绘制你的资产流动路径并标注每一步费用与时间。



### 第7课 合约交互基础与风险信号
- 目标：无需编程完成基础合约交互，能识别高危合约信号。
- 理论（简明）：ABI/函数与读写的区别；代理/可升级与 Owner 权限；renounce 的意义；Tax/Honeypot/黑名单机制；授权与 allowance 风险；源码验证与审计要点。
- 实操：在 Etherscan 的 Read Contract 读取余额或总供给；查看 Owner/代理合约与 LP 锁状态；不发起 `approve`。
- 工具：Etherscan、Revoke.cash、DeBank。
- 作业：完成一次 `read` 调用并用 5 点清单评估一个项目的真伪与风险。

### 第8课 Web3 赚钱机会地图（结合当前热点）
- 目标：建立可执行的赚钱路径与信息获取方法，避免盲目追高。
- 理论（简明）：
  - 路径：交易/Meme、稳健收益（稳定币/LP/质押）、空投/积分（Galxe/Zealy）、开源赏金（Gitcoin）、节点/再质押（EigenLayer）、DePIN/RWA、SocialFi、NFT、套利/做市。
  - 名词速查：Alpha、TVL、FDV、24h 量、滑点/深度/价差、授权/撤销、Rug/Honeypot/交易税、Renounce/LP 锁、MEV/Sandwich、Funding、Vesting 等。
  - 风险框架与执行：仓位与资金管理、流动性与周期、信息源验证（官方公告/研究报告/链上数据）、制定策略与复盘。
- 实操：选择一个标的/活动，查指标与规则；写 3 条参与准则与止损条件；不执行交易，仅完成“策略卡”。
- 工具：Binance Research/Academy/市场数据、Messari、Nansen、DeFiLlama、DexScreener、TradingView、Dune、L2Beat、项目官方公告与 GitHub。
- 作业：提交一页“策略卡”（目标、入口、指标、风控、执行与复盘）。
